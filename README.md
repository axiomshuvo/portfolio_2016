# Portfolio 2016

A personal portfolio website built in 2016 to showcase projects, skills, and contact information.  
This repository contains the full source code of a classic developer portfolio.

---

## Overview

This project is a simple, responsive portfolio website created using standard web technologies. It demonstrates front-end layout, styling, and basic server-side functionality.

The goal of the project was to create a clean personal site to present work and experience.

---

## Features

- Personal introduction section
- Project showcase
- Contact section
- Responsive layout
- Lightweight structure
- Simple PHP handling for downloads/pages

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP

## How to Run Locally

1. Clone the repository

```
git clone https://github.com/axiomshuvo/portfolio_2016.git
```

2. Navigate into the folder

```
cd portfolio_2016
```

3. Start a local PHP server

```
php -S localhost:8000
```

4. Open in browser:

```
http://localhost:8000
```

You can also open `index.php` directly in a browser for static viewing.
